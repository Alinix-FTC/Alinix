# Alinix

Align. Create. Innovate.

The Alinix landing page — a single static [index.html](index.html) with an animated intro:
a power-on gradient flare, the Alinix mark, and a lens-flare scan that reveals
**Align → Create → Innovate → Alinix**, with the logo mark serving as the "A" in each word.

No build step. Serve the repo root with any static host, or locally:

```sh
python3 -m http.server 4173
```
